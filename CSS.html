CASCADING STYLE SHEETS

USED FOR STYLE, LAYOUT OF WEB PAGES

CSS (Cascading Style Sheets) is a styling language used to describe the presentation and appearance of HTML elements on a web page. With CSS, you can control the colors, fonts, layouts, and other visual aspects of your website. It enables you to separate the content and structure of your web page from its design, making it easier to maintain and update. CSS offers a wide range of selectors, properties, and values to apply styles consistently across multiple web pages, resulting in a cohesive and professional-looking website design.

Note: HTML and JavaScript is also usually learned along with CSS.
